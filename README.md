# 🗡️ 鬼滅之刃 英語修行 — Demon Slayer English Quest

用《鬼滅之刃》劇情學英文閱讀與文法的互動闖關網站。共 **13 關**，主題為**鬼殺隊討伐十二鬼月（上弦＋下弦 12 鬼）與鬼王無慘**，每關涵蓋**兩種以上文法主題**，難度為**國中二～三年級（grade 8–9 / CEFR A2–B1）**。每關讀完劇情與閱讀短文後，從題庫**隨機抽 20 題（閱讀5＋文法5＋單字10）**，**限時 15 分鐘、全部答對**才能通關並解鎖下一關（時間到或答錯太多需重來）。

## ✨ 功能特色
- **13 關劇情**：下弦之伍累 → …下弦各鬼 → 上弦之陸墮姬妓夫太郎 → …上弦各鬼 → 鬼王無慘，依原作討伐順序推進
- **每關 2+ 文法主題**：現在式/進行式、過去式/過去進行、未來式/can-could、比較級/最高級、完成式/must-should、不定詞/動名詞、被動/關係代名詞、連接詞、條件句/附加問句、假設語氣/間接問句，後段並有時態/子句/全文法總複習
- **每關閱讀測驗**：對應劇情的英文短文（≤300字），測驗中固定考 5 題閱讀理解
- **每句配圖**：每關 10 例句各搭配一張 gpt-image-2 生成的鬼滅風插圖（共 130 張）＋13 張關卡封面
- **限時闖關**：每關 15 分鐘倒數，時間到沒做完就重來
- **AI 即時出題（選用）**：填入自己的 **OpenAI 或 Anthropic** 金鑰後，由 AI 依該關文法、單字與短文即時生成 20 題（閱讀5＋文法5＋單字10），無限不重複
- **音效**：Web Audio 合成的答對／答錯／通關／失敗音效（可開關，離線可用）
- **遊戲化**：愛心生命、XP 經驗值、進度條、通關畫面、自動存檔（localStorage）

## 🚀 使用方式
直接開啟 `index.html` 即可遊玩，或造訪 GitHub Pages 線上版。

### 啟用 AI 即時出題
1. 到 [console.anthropic.com](https://console.anthropic.com) → API Keys 取得金鑰
2. 點網站右上角 ⚙️ 設定，貼上金鑰、選擇模型，儲存
3. 進入任一關，開啟「🤖 AI 即時出題」開關

> 🔒 **隱私**：API 金鑰只會存在你自己的瀏覽器（localStorage），**不會上傳、不會進入此 repo**。金鑰未填或呼叫失敗時，會自動改用內建題庫。
