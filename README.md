# 🗡️ 鬼滅之刃 英語修行 — Demon Slayer English Quest

用《鬼滅之刃》劇情學英文閱讀與文法的互動闖關網站。共 **20 關**，每關鎖定**單一文法主題**，難度為**國中二～三年級（grade 8–9 / CEFR A2–B1）**。每關讀完劇情後，從題庫**隨機抽 5–7 題**，**全部答對**才能通關並解鎖下一關。

## ✨ 功能特色
- **20 關劇情**：依原作主線推進（賣炭日常 → 滅門 → 最終選別 → 鬼殺隊 → 柱 → 與無慘決戰）
- **單一文法主題／關**：現在式、過去式、未來式、完成式、比較級／最高級、助動詞、被動語態、不定詞／動名詞、關係代名詞、連接詞、條件句、間接問句、附加問句…
- **AI 動漫封面**：每關一張 gpt-image-2 生成的鬼滅風格場景插圖
- **AI 即時出題（選用）**：填入自己的 Anthropic API 金鑰後，由 Claude 依該關文法主題即時生成全新題目，無限不重複
- **音效**：Web Audio 合成的答對／答錯／通關／失敗音效（可開關，離線可用）
- **遊戲化**：愛心生命、XP 經驗值、進度條、自動存檔（localStorage）

## 🚀 使用方式
直接開啟 `index.html` 即可遊玩，或造訪 GitHub Pages 線上版。

### 啟用 AI 即時出題
1. 到 [console.anthropic.com](https://console.anthropic.com) → API Keys 取得金鑰
2. 點網站右上角 ⚙️ 設定，貼上金鑰、選擇模型，儲存
3. 進入任一關，開啟「🤖 AI 即時出題」開關

> 🔒 **隱私**：API 金鑰只會存在你自己的瀏覽器（localStorage），**不會上傳、不會進入此 repo**。金鑰未填或呼叫失敗時，會自動改用內建題庫。

## ⚠️ 版權聲明
《鬼滅之刃》角色與世界觀版權屬原作者**吾峠呼世晴**及集英社。本專案為**非商業的英語教學練習**用途，封面為 AI 生成之原創風格場景，不含官方素材。

純前端單檔（HTML/CSS/JS），無需後端建置。AI 出題透過瀏覽器直連 Anthropic Messages API（`anthropic-dangerous-direct-browser-access`）。
